# ①課題番号-プロダクト名
恋人をのろけるwebアプリをアップデートしました

## ②課題内容（どんな作品か）
-恋人をのろけるアプリです。
-メニューの「ノロケる♡」からデータを登録します。「登録したデータはノロケを見る」からスライドで確認できます。
-「プロフィールを入力する」からデータを登録すると「プロフィール」から自分のプロフィールが確認できます。
-トップページではデータを6件取り出すようにしました。
